# P0 / P2 테스트 시나리오

## P0 서버권한 정합성

1. **로그인 후 플랜 우선순위**
   - 사용자 `user_subscriptions`에 `plan='pro'` 등록.
   - 앱 재접속 시 설정 라벨이 `Pro`로 보이고 `symbolPlan`/`planFeatures`가 Pro로 반영되는지 확인.

2. **구독 비활성화 시 즉시 Free 복귀**
   - `user_subscriptions.status`를 `cancelled` 또는 만료된 상태로 변경.
   - 앱 재접속 시 또는 즉시 refresh 시 Free로 전환되는지 확인.

3. **오프라인/로컬 변조 차단**
   - 로컬 `localStorage`에 `plan=pro`를 임의 설정.
   - DB 메타/구독이 free면 앱이 Free로 노출되는지 확인.

## P2 기록 생산성 번들

4. **활동 템플릿**
   - `템플릿 저장`으로 `💻 2h` 입력 후 템플릿 패널에서 한 번 클릭.
   - 해당 날에 `💻` 합산 시간이 2h씩 반영되는지 확인.

5. **Todo 반복 규칙**
   - 월~금(`Mon~Fri`) + `2w` 선택 후 Todo 추가.
   - 2주 내 각 요일에 동일 제목이 중복 없이 생성되는지 확인.

6. **시간 범위 입력 + 자동 계산**
   - `2:25 + 2h` 입력 시, 표시 시간이 `2h 25m`로 저장되는지 확인.
   - 시작/종료 시간 `14:00` + `2h` 입력 시 종료가 `16:00`으로 자동 계산되는지 확인.
   - 범위 `14:00 - 12:00` 입력 시 24시간 초과 구간으로 처리되는지 확인.

7. **다중 기기 동시 수정 감지**
   - A 디바이스에서 같은 날짜 항목 수정 후 저장.
   - B 디바이스에서 같은 날짜를 먼저 로드한 상태로 바로 수정 시 충돌 알림이 표시되는지 확인.
